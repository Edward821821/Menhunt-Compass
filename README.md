# Menhunt-Compass
## Installing
1. put this datapack in your datapacks folder of your world folder _(maybe one day Modrinth does it for you...)_ if you are using the mod version, do what you do with mods
2. type **/reload** into the chat (no need to restart the world ro server)
3. type **/tag <name> add runner** into the chat and replace _<name>_ with the name of the player who should be hunted
4. type **/function menhunt:give** into the chat to get each time a compass (hand them out to the players who hunt)

## Important to know
- Compass updates by rightclicking
- Compass shows the location of entering the portal when the hunted player is in another dimension (but no custom dimensions!)
- all commands named need operator level 2 to be executed
